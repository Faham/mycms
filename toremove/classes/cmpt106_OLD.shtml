CMPT 106.3 - Design and Construction of Games and Interactive Systems
CMPT 106 continues where 105 leaves off, and introduces additional concepts in computer science and interaction design. The course will focus on the design and analysis of a wide variety of computer games. You will practice your programming skills (and learn new ones!) by designing and implementing your own games.  CMPT 106 will teach you practical computing skills while allowing you to exercise your creativity and design skills.

You will learn new computing topics such as representation of data, use of expressions, development of subprograms, and creation and manipulation of graphical images. Design topics will include consideration of narrative, game rules, collaboration, and animation.


Learning Objectives
By successfully completing the course, a student should be able to:
  • Discuss game design features and game mechanics
  • Relate current game designs to earlier examples, from the direct predecessors to the first recorded examples
  • Plan game design projects according to best practice descriptions
  • Develop a game design concept from initial idea to a full game design document, using iterative design processes and prototyping
  • Implement your game concept and create an interactive computer game using new programming skills
  • Specify target audiences and develop specific game design concepts for specific target audiences
  • Analyze different game designs using analytical tools to be able to suggest design changes and compare different game designs
  • Pitch game design concepts for an audience
